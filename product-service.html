VESTALIS CLOUD

Vestalis cloud has been designed and developed by SRETT to manage all data sent by IoT. This architecture is successfully used throughout SRETT Medical to manage tens of thousands of patient data every day with strict confidentiality and security protocol.

The same proven architecture is now being used for the Industry. Clients can access Vestalis cloud and download data directly using Optimum Vision or using an API so that data can be used immediately throughout their own applications.

OPTIMUM VISION

Optimum Vision will provide a simple and efficient interface to visualise all devices currently under use. For each device, it will enable the Operator to follow the value and set alarms for each device.
